## 开始写作

要添加新文章，只需在 note 目录下创建 `.md` 文件，然后运行：

```bash
# 更新侧边栏
node generate-sidebar-structure.js

# 更新首页
python init.py
```

创建其他目录请参考项目 [README.md](https://github.com/ZhuBaiwan-oOZZXX/PurePage) 文件。

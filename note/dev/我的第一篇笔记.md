    """计算斐波那契数列"""
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# 使用示例
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")

## 🚀 博客使用指南

### 添加新文章
1. 在 `note/` 目录下创建 `.md` 文件
2. 使用上述 Markdown 语法编写内容
3. 运行生成脚本更新侧边栏和首页

### 更新脚本
```bash
node generate-sidebar-structure.js
```

### 部署流程
```bash
# 提交更改
git add .
git commit -m "添加新文章"

# 推送到 GitHub
git push origin main
```

GitHub Pages会自动部署，几分钟后即可访问新内容。
